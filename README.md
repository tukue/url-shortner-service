# URL Shortener Service

A URL shortening service built with Node.js and Express.

## Project Structure
```
url-shortener-service/
├── src/
│   ├── services/
│   │   └── urlShortener/
│   ├── repositories/
│   ├── models/
│   ├── config/
│   ├── controllers/
│   └── routes/
├── tests/
├── package.json
└── README.md
```

## Setup
1. Install dependencies: `npm install`
2. Configure environment variables
3. Run the service: `npm start`

## Testing
Run tests with: `npm test`
